(9/12/20 - 22/12/20) User Centered Design recommendation for Private Beta

The following flows are based upon understanding of user needs identified and designed for over the alpha and beta phase.

  1. View stand alone service starting point
  2. View stand record with no documents (Unhappy path)
  3. View stand alone record with documents
  4. Start from ePR (Electronic Public Register) - go to documents section if requested

Searching across multiple permits

Research has so far suggested that this is not a useful feature and there is evidence indicating that users' ability to complete key tasks are impaired by having this option available to them.

The recommendation is to leave this feature out of private Beta and actively monitor whether incoming requests could have likely been satisfied by the feature being available. Its inclusion in the service should be reevaluated at the conclusion of private Beta.
